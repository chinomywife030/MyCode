# 🧪 登入/註冊頁條款文字測試步驟

## ⚠️ 如果看不到更新，請按照以下步驟操作：

### 步驟 1：重啟開發服務器

在終端機中：

```bash
# 停止當前的開發服務器（Ctrl+C）
# 然後重新啟動
cd bangbuy
npm run dev
```

---

### 步驟 2：清除瀏覽器緩存

#### 方法 A：硬刷新（推薦）
- **Windows/Linux**: `Ctrl + Shift + R` 或 `Ctrl + F5`
- **Mac**: `Cmd + Shift + R`

#### 方法 B：開無痕視窗
- **Chrome**: `Ctrl + Shift + N` (Windows) 或 `Cmd + Shift + N` (Mac)
- **Firefox**: `Ctrl + Shift + P`

#### 方法 C：手動清除緩存
1. 打開開發者工具（F12）
2. 右鍵點擊刷新按鈕
3. 選擇「清除緩存並硬性重新載入」

---

### 步驟 3：訪問登入頁並驗證

訪問：`http://localhost:3000/login`

---

## ✅ 預期結果

### 📝 **註冊 Tab（signup）**

應該顯示：

```
註冊即表示您已閱讀並同意《使用條款》、《隱私權政策》與《免責聲明》。
```

**檢查項目：**
- [ ] 文字開頭是「**註冊**即表示」（不是「登入」）
- [ ] 包含「已閱讀並」
- [ ] 包含三個連結：《使用條款》、《隱私權政策》、《免責聲明》
- [ ] **沒有 checkbox**
- [ ] 文字是灰色小字（`text-xs text-gray-600`）
- [ ] 位於註冊按鈕上方
- [ ] 註冊按鈕是藍色且可點擊

---

### 🔑 **登入 Tab（login）**

應該顯示：

```
登入即表示您同意《使用條款》與《隱私權政策》。
```

**檢查項目：**
- [ ] 文字開頭是「**登入**即表示」
- [ ] **沒有**「已閱讀並」
- [ ] 只有兩個連結：《使用條款》、《隱私權政策》
- [ ] **沒有**《免責聲明》
- [ ] **沒有 checkbox**
- [ ] 文字是灰色小字
- [ ] 位於登入按鈕上方
- [ ] 登入按鈕可點擊

## 🔍 如果還是沒改，請檢查：

### 檢查 1：確認檔案已儲存
打開 `bangbuy/components/LegalConsentBlock.tsx`，第 31 行應該是：

```typescript
{isRegister ? '註冊' : '登入'}即表示您{isRegister ? '已閱讀並' : ''}同意
```

### 檢查 2：確認 Next.js 編譯成功
終端機中應該顯示類似：

```
✓ Compiled successfully
```

如果顯示錯誤，請先修復錯誤。

### 檢查 3：確認沒有其他 LegalConsentBlock 組件
在專案中搜索是否有其他同名組件：

```bash
# Windows PowerShell
Get-ChildItem -Path . -Filter "LegalConsent*" -Recurse
```

### 檢查 4：確認瀏覽器開發者工具沒有錯誤
1. 按 F12 打開開發者工具
2. 查看 Console 是否有紅色錯誤
3. 如果有錯誤，請截圖並回報
