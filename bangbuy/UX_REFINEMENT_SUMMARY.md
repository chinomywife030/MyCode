### 2. **優化 ModeToggle：明確可點擊的 Tabs**

**檔案**：`bangbuy/components/ModeToggle.tsx`

#### 設計改進
**之前**：看起來像狀態指示燈（圓形 pill + 圓點 emoji）
**現在**：明確的 Tabs 設計（方形分段按鈕 + 清晰 icon）

#### 新設計特點
```tsx
// 外框：白底 + 邊框 + 陰影
className="bg-white border-2 border-gray-200 shadow-sm"

// Active 狀態：實心背景 + 內陰影
買家：bg-blue-500 text-white shadow-inner
代購：bg-orange-500 text-white shadow-inner

// Inactive 狀態：白底 + hover 有色
買家：bg-white hover:bg-blue-50 hover:text-blue-600
代購：bg-white hover:bg-orange-50 hover:text-orange-600

// Icon
買家：🛒（購物車）
代購：✈️（飛機）
```

#### 尺寸
- **Compact（Navbar）**：`minHeight: 44px, minWidth: 85px`
- **Full（其他）**：`minHeight: 48px, minWidth: 110px`

#### 無障礙
- `role="tablist"` / `role="tab"`
- `aria-selected={mode === 'requester'}`
- `cursor: pointer`

## 📝 設計原則總結

### 1. 命名語意化
- **買家**：我要買東西、發布需求、找人幫忙
- **代購**：我要接單、幫人代買、賺取收入

### 2. 切換明確化
- 不是狀態指示燈，是身份切換
- 像 Tabs，不是 Toggle
- 一眼可辨當前身份（文字 + 顏色 + icon）

### 3. 平台成熟感
- 避免過小或隱晦的控制元件
- 避免玩具感 UI（過多圓形、過多動畫）
- 行動裝置優先（拇指可輕鬆點擊）

### 4. 文案簡潔有力
- 去除冗長描述
- 直接傳達價值主張
- 強調行動力（「發布」「接單」「賺錢」）
